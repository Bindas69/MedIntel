# Fix FCM Service Issues

## Identified Problems

### FCM Service Code Issues (lib/services/fcm_service.dart)
1. **Missing import** (line 1): No import for `firebase_options.dart` but background handler calls `Firebase.initializeApp()` without options
2. **Background handler Firebase initialization** (line 8): Calls `Firebase.initializeApp()` without options - will fail with newer Firebase SDK
3. **Deprecated API usage** (line 43-44): `resolvePlatformSpecificImplementation` approach is still supported but can be improved

### Android Configuration Issues (android/app/build.gradle.kts)
4. **Firebase Messaging not enabled** (line 57): `firebase-messaging` dependency is commented out
5. **Missing FCM permissions** in AndroidManifest.xml: No `POST_NOTIFICATIONS` permission for Android 13+, no Firebase messaging service

## Fixes Required

### Fix 1: FCM Service - Add Import (line 1-4)
Add import for `firebase_options.dart`:
```dart
import 'package:med_intel/firebase_options.dart';
```

### Fix 2: FCM Service - Background Handler (line 6-9)
Change:
```dart
Future<void> _firebaseMessagingBackgroundHandler(RemoteMessage message) async {
  await Firebase.initializeApp();
  debugPrint('FCM background message: ${message.messageId}');
}
```
To:
```dart
@pragma('vm:entry-point')
Future<void> _firebaseMessagingBackgroundHandler(RemoteMessage message) async {
  await Firebase.initializeApp(options: DefaultFirebaseOptions.currentPlatform);
  debugPrint('FCM background message: ${message.messageId}');
}
```

### Fix 3: Android build.gradle.kts (line 57)
Uncomment the Firebase messaging dependency:
```kotlin
implementation("com.google.firebase:firebase-messaging")
```

### Fix 4: AndroidManifest.xml - Add Permissions
Add inside `<manifest>` tag:
```xml
<uses-permission android:name="android.permission.POST_NOTIFICATIONS" />
```

## Implementation Steps

1. Update `lib/services/fcm_service.dart`:
   - Add import for `firebase_options.dart`
   - Add `@pragma('vm:entry-point')` annotation to background handler
   - Update `Firebase.initializeApp()` to use `DefaultFirebaseOptions.currentPlatform`

2. Update `android/app/build.gradle.kts`:
   - Uncomment `firebase-messaging` dependency

3. Update `android/app/src/main/AndroidManifest.xml`:
   - Add `POST_NOTIFICATIONS` permission for Android 13+ support
